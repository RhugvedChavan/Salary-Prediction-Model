# Salary-Prediction-Model

# Objective :-

To develop a machine learning model that predicts salary.


# Summary :- 

1. Developed a machine learning model to predict employee salaries based on input features.
2. Used the Linear Regression algorithm for predictive analysis.
3. Collected and preprocessed salary-related data to improve model performance.
4. Trained the model using historical salary records and tested it on unseen data.
5. Generated salary predictions that can assist in recruitment and compensation planning.


# Key Features :- 


1. Predicts salary based on employee attributes such as experience, education, and skills.
2. Simple and interpretable Linear Regression model.
3. Data preprocessing including handling missing values and feature selection.
4. Visualization of relationships between salary and independent variables.
5. Fast prediction with minimal computational requirements.


# Results :- 


1. Successfully trained and tested the salary prediction model.
2. Achieved accurate salary estimates on the test dataset.
3. Identified a positive relationship between experience and salary.
4. Reduced manual effort in salary estimation.
5. Generated prediction outputs that closely matched actual salary values.


# Conclusion :- 


1. Linear Regression is effective for salary prediction when a linear relationship exists.
2. The model provides quick and reliable salary estimates.
3. Data quality significantly impacts prediction accuracy.
4. The project demonstrates practical application of machine learning in HR analytics.
5. Future improvements can include advanced algorithms for higher accuracy.


# Technologies Used :- 

python Libraries - 

1. Pandas (Data Manipulation)
2. NumPy (Numerical Computation)
3. Scikit-learn (Linear Regression Model)
4. Matplotlib / Seaborn (Data Visualization)
